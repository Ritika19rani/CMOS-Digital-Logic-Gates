# Description 
 This repository contains the design and implementation of CMOS digital logic gates using Cadence Virtuoso with GPDK90(90nm CMOS technology).
 The repository includes the schematic, layout and corresponding design files for both basic and complex logic gates. 
 These designs were created to understand CMOS circuit implementaion, layout esign and VLSI design flow.
 # Contents
 - 2-input AND Gate
 - 2-input OR Gate
 - 2-input XOR Gate
 - 2-input XNOR Gate
 - 2-input NOR Gate
 - 2-input NAND Gate
 - NOT(inverter)
 - 8-bit Logic Gtae Design
 - Schematic Diagram
 - Layout Design
# Tools Used
- Cadence Virtuoso
- GPDK90(90nm CMOS technology)
